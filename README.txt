Allow only members of your site to view certain posts/pages on your site. If a user is not logged in they will be shown a  modal with a content restricted message.

== Description ==
The plugin was created to incentivize your visitors to create an account on your website. Notification will advise users they must be logged into view the content of your choosing. In settings you can edit the animation, colors and text of the notification your users will recieve. 

== Installation ==
1.  Upload the plugin to the /wp-content/plugins/ directory or use the Add New feature
2. Activate the plugin through the ‘Plugins’ menu in WordPress
3 Go-to a page or post and add the shortcode [visitor-content-wall]
